# 🍃 无印极简与空谷呼吸感 (Organic Minimal & Muji Calm)

拒绝过度的饱和度轰炸，摒弃一切无谓的光影和喧嚣。这是为生活方式、人文反思、极简健康类 APP 或者“侘寂风”（Wabi-Sabi）茶艺品牌定制的极奢视觉。在这里，“没有设计”并且释放巨大的留白，就是最顶级的奢侈排版。任何试图填补空白的尝试都会降低这一格调。

## 📍 :root 色彩留白信仰 
全场不能有任何色彩的饱和度超过 20%。你将进入极具禅意的超低反差色板：

```css
:root {
  /* 如同宣纸、亚麻布一样具有呼吸感的原生大地基调 */
  --bg-zen: #F2F0EB; 
  --bg-sand: #EAE6DF;
  
  /* 最深也只用到极具克制感的茶褐黑与灰橄榄绿 */
  --text-wabi: #3A3530; 
  --text-muted: #8E8A83;
  --accent-olive: #68735C; /* 极其微弱的一点低饱和度自然色 */

  /* 彻底消除所有边界与生硬块面 */
  --border-fine: 1px solid rgba(142, 138, 131, 0.2); /* 浅若无痕的边框线 */
}
```

## 📍 宋体之禅与空气排版 (Zen Typography)
- 如果可以获取中文字体，**必须大量使用极细的宋体/明体**。在英文中则请使用极端素净消瘦的衬线体诸如 `Cormorant Garamond`, `Lora`, 或者 `Georgia`。
- 这是唯一一个**不要拉伸出巨型标题**的方案。最大的标题应该维持克制的 `2.5rem` 并采用如 `letter-spacing: 0.1em;` 的巨大字间距散开来！字重维持在精细的 `font-weight: 300` 甚至 `200`（如果有）。
- 正文行高达到近乎夸张的 `line-height: 2.0;`，让人在阅读时感受到清风拂面的宁静！

## 📍 极致寂静的大留白空间 (Void Spaces)
- 不要让元素贴着边界。
- 把你的卡片和图表当作画框中心的一叶孤舟，请保留多达 40% 甚至 50% 的外部 `padding` 以及内间距。即使只有寥寥数字的居左说明，也要让它孤芳自赏。
- 图片与内容之间应当加入巨大的 `margin`，营造“孤寂感”。

## 📍 CSS 水墨流连的数据渲染 (Fading Info Visualization)
- 不要强烈的柱状图！这里所有用于图示化的折线图或者进度条必须极其精细纤细：
  进度条甚至可以是 `1px` 高的一条虚线或实线，其前端用一粒 `--accent-olive` 的微小圆点示意。
- 数据可以配合不显眼的微渐变背景 `linear-gradient(to right, rgba(104, 115, 92, 0.15), transparent)` 在画布中默默消隐。
- 这是一份让人屏息阅读的高尚读物报告。
