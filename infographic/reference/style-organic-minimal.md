# Ethereal / Organic 莫兰迪自然无印呼吸风 (Organic Minimal)

不要刺眼色彩、不要强力冲击，这里需要极高负空间的温婉空灵和治愈系的高雅审美。多用于人文社科、自然健康。

## 📍 核心视觉法则 (Aesthetics)
- **空气软壳 (Soft Air)**：整个画布的色温要压在很克制、非常温润的莫兰迪低饱和色盘里。最爱用 枯叶褐 `#C1A68D`、深林茶绿 `#7D8C70` 或 软泥灰 `#EBDFD3`。永远不要出现 `#000` 纯黑正文文字。
- **禅意渗透设色 (Zen Bleed)**：所有的卡片如果没有必要，绝对不要边框线 `border`，所有边界最好是通过极弱的光晕交叠暗示，或者压根就是纯底色融合。
- **诗意古典排印 (Poetic Serif)**：让大标题充满情感与力量！必须通过 Google Fonts 强行引入极度优雅浪漫的高定衬线体（强制 `Cormorant Garamond` 或 `Playfair Display`）。而在其下方承接的小片精巧注释则应该切回干净圆滑的无衬线体（`Outfit` 或 `DM Sans`），刚柔并济。

## 🛠 原生 CSS 数据绘图大招 (CSS Visaul Hacks)
你不能用任何生硬的直线数据表破坏这种呼吸感：
- **无边云雾虚化**：如果你想把两块数据切开，就只给其中一块加上大得夸张又极度若有若无的 `box-shadow: 0 30px 60px rgba(0,0,0, 0.03)`，不露痕迹。
- **纯粹圆锥占比环 (Conic Gradients)**：圆润、没有任何刺边的数据饼图：直接由原生 CSS 的 `conic-gradient(#C1A68D 0% 64%, rgba(193, 166, 141, 0.15) 64% 100%)` 构建优雅的双色占比图。
- **悬停游离留白空间 (Negative Space)**：大面积甚至令人疯狂的“空出来”。比如画布的上方留出整整 `padding-top: 200px` 纯色，只是左下角飘起一段极美的文字诗眼与单个震撼数字。
