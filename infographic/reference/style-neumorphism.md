# Neumorphism 极致零接缝微拟物 (Soft UI)

放弃所有的扁平化！这是一场基于高维度坐标系打造的硅胶质感科技面板重塑，如同真实的物理凸起开关一般温润。

## 📍 核心视觉法则 (Aesthetics)
- **同色系绝对融合 (Unified Tonal)**：不要随便用互补色！画布背景色、所有卡片的底色，必须是 极度精确的同一颜色！比如柔和微亮的科技白 `#E0E5EC` 或者 深海潜舰灰 `#1a1b1e`。
- **神级光源算力 (Dual Shadows)**：任何组件如果要凸出来或者凹进去，靠的是你精湛的光影：一侧高光，另一侧深渊。
  - **凸起面板 (Convex)**：`box-shadow: 9px 9px 16px rgb(163,177,198,0.6), -9px -9px 16px rgba(255,255,255, 0.5);` 
  - **凹陷插槽 (Concave)**：`box-shadow: inset 6px 6px 10px 0 rgba(163,177,198, 0.7), inset -6px -6px 10px 0 rgba(255,255,255,0.8);`
- **中性高级排版 (Neutral Grotesk)**：排字不要抢戏。字体推荐使用最包容无害的圆滑科幻无衬线体 `Space Grotesk` 或者极客圆体 `Nunito`。

## 🛠 原生 CSS 数据绘图大招 (CSS Visaul Hacks)
这要求你的 CSS 技法具备近乎全息摄影级别的掌控力。
- **下沉式能量刻度板 (Inset Tracks)**：进度条的最外层容器，使用 **凹进去的内阴影 (inset)** 变成一个沟槽；而里面实际填充进度的实色条 `div` 则是一个带着小发亮高光边缘的实体条。
- **浮动旋钮图表 (Knob Doughnuts)**：画一个绝对完美的正圆形（`border-radius: 50%`），给它设置 **凸起的阴影组合**，内部用一块小一点的内凹正圆形去挖出中空效果！这就成了一个实体的 3D O-Ring 环形图外壳，配合 `conic-gradient` 的圆边剪切效果，这胜过任何 2D 图表引擎！
- **按压光效质感 (Press State Toggles)**：那些想要着重显示的核心 KPI 面板周边，可以用类似开关被按下去发散的微小圆型按压 `box-shadow` 做底垫。
